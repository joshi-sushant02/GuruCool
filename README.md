# GuruCool A Virtual Classroom
This is an attempt to clone the best features of google classroom using django.



## Features Included 
- Custom Admin dashboard
- Create Classroom
- Join Classroom
- Add Posts 
- Create Assignments 
- Grade Assignments 
- Add Resources 
- Courses 
- Forgot password 
- User registration
- Much more...
